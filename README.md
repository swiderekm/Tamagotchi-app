## About The Project

This is my **school project** Tamagotchi game — a digital pet simulation where you take care of a cute creature by feeding it, playing with it and putting it to sleep.

The main learning goals of the project were:

- Understanding and applying **Object-Oriented Programming** (OOP)
- Practicing good code **structure & organization**
- Creating clean separation between game logic, UI and data
- Writing readable and maintainable code

## Features

- Real-time needs decay (hunger, boredom, energy)
- Different actions: Feed 🍎 • Play 🎾 • Sleep 😴
- Activity log with all the things that happened stored

  
## Technologies
- HTML5
- CSS3 (custom styling, no framework)
- JavaScript (axios + DOM manipulation)
- randomuserme – public free API ([API link]([url](https://randomuser.me/api/0.8)))
